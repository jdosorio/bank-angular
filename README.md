# BankAngular

Prueba tecnica generada con Angular en su version 17.1.3.

# Instalar Dependencias

Correr `npm i`

## Correr proyecto

Correr `npm run start` 

## Construir Proyecto

Correr `ng build`

## Correr test unitarios

Correr `npm test`.

## Correr test unitarios (Coverage_

Correr `npm run test:coverage`.
