# BankAngular

Prueba tecnica generada con Angular en su version 17.1.3. Se abordo la prueba de manera sencilla, sin usar frameworks del lado del front, usando nuevos aspectos de angular como signals, pero tambien usando comunicacion de componentes mediante @Input y @Output. Se cubrieron los diferentes requerimientos y se agrego una animacion en la parte superior usando Lottie. Se agrego paginador y se busco hacer el dise√±o responsive.

# Instalar Dependencias

Correr `npm i`

## Correr proyecto

Correr `npm run start` 

## Construir Proyecto

Correr `ng build`

## Correr test unitarios

Correr `npm test`.

## Correr test unitarios (Coverage_

Correr `npm run test:coverage`.
